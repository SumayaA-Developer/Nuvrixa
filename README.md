# Nuvrixa

Nuvrixa is a frontend-only Next.js marketing website for AI automation discovery, workflow consulting, and business process improvement.

## Phase 1 Status

Phase 1 establishes the production foundation for the website.

Completed:

- Next.js App Router setup
- TypeScript setup
- Tailwind CSS v4 setup
- Geist font setup
- Nuvrixa dark/aqua global styling
- SEO metadata foundation
- Robots and sitemap routes
- Professional folder architecture
- Vercel-ready configuration

## Local Development

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Build Rule

This is intentionally a frontend-only website. No backend, no database, and no server-side business logic should be added unless the project scope changes.
